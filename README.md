# AccelerateRT
Explore acceleration structures for ray tracing

------

### Dependencies

```
GLFW
ModernGL
```
To install dependencies for local, activate project:
```julia
julia> ]
(@v1.7) pkg> activate .
```

To execute, run:
```
julia --project=. main.jl
```

Can optionally install dependencies globally, and run `main.jl` directly.
